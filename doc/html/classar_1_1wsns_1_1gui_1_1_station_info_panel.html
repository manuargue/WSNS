Detailed Description

A panel that displays information related to a particular station.

This class has a reference to the selected station, which can be changed by the function setStation. The panel has three sections: the title, which shows the station name, its ID and a picture of the station; the connection list, which shows all the nodes connected to the selected station; the sensor list, which shows the sensors connected to the station and allow you to add, remove or reset any sensor of the list.
