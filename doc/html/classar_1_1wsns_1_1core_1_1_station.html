Packet ar.wsns.core.Station.handlePacket ( Packet  packet)

Determines the behavior of this station when it receives a packet.

If this station is not active, returns null. If this station is not the destination of the packet, forwards it to the destiny station. Otherwise, returns a new packet containing the sensors values.

Parameters:
packetPacket received
Returns:
The packet response, if this station is active; otherwise null
