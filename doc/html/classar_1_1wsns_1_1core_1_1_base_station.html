void ar.wsns.core.BaseStation.updateDataBase ( )

Creates a SQL table for each station in the network.

Creates a SQL table for each station in the network. The name of the table is the concatenation of the word "station" and the station ID. If the table already exists, it is discarded and recreated. The the fields of the table are: * sensor: sensor type, String * date: date on which the data was stored, SQL Date * max, average and min: maximum, average and minimum daily
